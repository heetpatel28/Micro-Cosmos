### Health Check

### Items API

```
GET    /api/items        - Get all items
GET    /api/items/:id    - Get item by ID
POST   /api/items        - Create new item
PUT    /api/items/:id    - Update item
DELETE /api/items/:id    - Delete item
```

### Example Request

```bash
# Create item
curl -X POST http://localhost:{{PORT}}/api/items \
  -H "Content-Type: application/json" \
  -d '{"name": "Example Item", "description": "Item description", "price": 29.99}'

# Get all items
curl http://localhost:{{PORT}}/api/items

# Get item by ID
curl http://localhost:{{PORT}}/api/items/{id}

# Update item
curl -X PUT http://localhost:{{PORT}}/api/items/{id} \
  -H "Content-Type: application/json" \
  -d '{"name": "Updated Item"}'

# Delete item
curl -X DELETE http://localhost:{{PORT}}/api/items/{id}
```
